This leads to rejection of the null hypothesis.

Permutation test

  • Randomly assign measurements to individuals.
  • Fit linear model.
  • Calculate the F-statistic.
  • Repeat \(N\) times.

\[ H_0:\text{ All measurements come from the same distribution} \\ H_1:\text{ The measurements are not from the same distribution} \] Reject \(H_0\) if the F-statistic of the original data is larger than \(95\%\) of F-statistics obtained from permuting the data.

Implementation of the permutation test

permTest <- function(df) {
  perm_df <- tibble(
    meas = df %>%
      pull(meas) %>%
      sample(),
    pers = df %>%
      pull(pers)
  )
  lm(log(meas) ~ pers, data = perm_df) %>%
    glance() %>%
    pull(statistic)
}

Conducting the permutation test

permTest_df <- tibble(
  run = seq(1, 999),
